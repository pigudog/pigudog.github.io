Linux-实操篇-3.1 常用基本shell命令

Created2022-11-10|Updated2022-11-11|computerLinux
|Post View:

Shell可以看做是一个命令解释器，为我们提供交互式的文本控制台界面。我们可以通过终端控制台来输入命令，由shell进行解释并最终交给内核执行。本章将分类介绍常用的基本shell命令

Linux是源于unix，unix的解释执行的是Bourrl shell,然后Linux的使用的是Bourrl again shell -> bash

Ubantu为了简化就是用dash，因为是从Debian来的

但是我们使用的是bash

帮助命令

提供了非常方便的命令手册，man-> manual
man + 命令名称

一个疑惑->man cd 会进入bash的解释，cd是shell的内置命令，就是所谓的

  • built-in 命令，一部分基础功能的系统命令是直接内嵌在shell中的，常驻系统内存中
    • cd
    • exit
    • 但是我们可以使用help命令， 获得shell内置命令的帮助信息help cd,注意没法help+外部命令
  • 外部命令

那么怎么判断命令的类型呢?



p–posix,ieee定义的标准命令
我们可以输入 cd 1p cd查看

reset=把整个bash直接初始化
善于使用tab键

print working directory 打印工作目录

  • 基本与语法：pwd 显示当前工作目录的绝对路径

r: recursive
也就是说如果我们复制某一个目录，但是在该目录下面有其他的子目录，会全部都被复制过去

rm

  • 删除文件或目录
  • rm [option] deleteFile 递归删除目录中所有内容,但是如果不加-r是删除文件，不能删除目录
  • -r 递归删除目录中所有内容
  • -f 强制执行删除操作，而不提示用于进行确定force
  • -v 显示指令的详细指向过程
  • -rf非常强大一下子全删干净了，删库跑路
    • rm -rf /神奇的指令
    • rm -f ./一个点表示当前目录

mv

移动文件与目录或重命名

如果后边给的是文件名，则移动过去之后还会重命名

  • man类似的翻页操作，空格翻到下一页，翻到最后即退出，回车则一行一行地往下翻，按b往前
